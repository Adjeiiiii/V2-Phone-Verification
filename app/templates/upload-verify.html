CSV Format Instructions

Please upload a CSV file with four columns in this order:

  1. first_name
  2. last_name
  3. email
  4. phone_number

Example:

first_name,last_name,email,phone_number
John,Doe,john@example.com,+12025551234
Jane,Smith,jane@example.com,+12025554321

Make sure your phone numbers include country code if needed (e.g., +1 for US).
